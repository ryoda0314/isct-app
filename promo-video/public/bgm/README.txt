BGM・効果音の置き場所
======================

ここに BGM 音源（mp3/wav）を置き、src/PromoVideo.jsx の BGM に設定するとループ再生されます。
  const BGM = 'bgm/track.mp3';   // 例
音量は BGM_VOLUME（既定 0.14＝控えめ）、フェードは BGM_FADE で調整。

──────────────────────────────────────────────
ずんだもん／ゆっくり解説で定番の「無料・商用利用可」素材サイト
（※必ず各サイトの利用規約を確認。クレジット要否はサイトごとに違う）

▼ BGM
- DOVA-SYNDROME   https://dova-s.jp/      … 13,000曲超。解説動画の定番。曲ごとに規約確認
- 魔王魂          https://maou.audio/     … 商用OK・改変OK。BGMもSEも豊富
- 甘茶の音楽工房   https://amachamusic.chagasi.com/  … 落ち着いた曲が多い
- PeriTune        https://peritune.com/   … ジャンル別に整理された高品質BGM

▼ 効果音（テロップの「ポンッ」や場面転換など）
- 効果音ラボ      https://soundeffect-lab.info/  … クレジット不要・商用無料。解説動画SEの鉄板
- 魔王魂          https://maou.audio/     … SEも配布

  BGM: DOVA-SYNDROME
