効果音(SE)の置き場所
======================

ここに短い効果音を置き、src/PromoVideo.jsx の SE に設定すると自動で鳴ります。
  const SE = {
    transition: 'se/transition.mp3',  // 章の切り替わり（場面転換）
    telop:      'se/pop.mp3',          // テロップ表示時（ポンッ）
  };
音量は SE_VOLUME（既定 0.5）。

鳴るタイミング（自動）:
  - transition … 章（##見出し）が変わる最初のセリフで1回
  - telop      … テロップ箇条書きが新しく出た時に1回

──────────────────────────────────────────────
おすすめ素材（クレジット不要・商用無料）
- 効果音ラボ  https://soundeffect-lab.info/
    ・テロップ用「ポンッ」: 「ボタン・システム音」「ポップな効果音」あたり
    ・場面転換用「シュッ/ワイプ」: 「アニメ・約物」カテゴリ等
- 魔王魂      https://maou.audio/   （SEあり）

ダウンロードしたファイルをここに置き、上記 SE にファイル名を設定してください。
未設定（null）なら無音で安全にレンダリングされます。

クレジット例（概要欄）: 効果音: 効果音ラボ
