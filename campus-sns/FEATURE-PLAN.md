# チャット機能拡張プラン

## 高優先度

### 1. メンション (@通知)
- 投稿・コメントで `@名前` と入力 → 対象ユーザーに通知
- 入力中にコースメンバーからサジェスト表示
- メンションされたテキスト部分をハイライト表示
- 通知テーブルに mention タイプ追加

### 2. 未読バッジ
- DM・チャットの未読メッセージ数をナビアイコンに表示
- 通知の未読数もバッジ表示
- Supabase Realtime で新着メッセージ検知
- 画面を開いたら既読に更新

### 3. 投稿検索
- タイムライン内のキーワード検索バー
- 投稿テキスト・投稿者名で絞り込み
- タイプフィルター（質問/資料/情報/etc）
- 検索結果ハイライト

### 4. 投稿ピン留め
- コース内の重要投稿を上部固定
- 投稿者本人 or 管理者がピン設定/解除
- ピン留め投稿は最大3件まで
- タイムライン上部に専用セクション
