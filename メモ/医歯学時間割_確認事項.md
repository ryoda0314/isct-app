# 医歯学時間割 確認事項

## 飯島さんに確認が必要な点

### 「顔面・顎・口腔疾患」(DEN.A309 / 021181) と「顎口腔医療」(DEN.A308 / 021180) の重複
- シラバス上、この2科目が**全く同じ日時**に登録されている（全27回すべて一致）
- 実際の授業はどうなっているのか？
  - 同じ授業を2科目コードで登録している？
  - 選択でどちらか一方だけ受ける？
  - 交互に行われる？
  - シラバスの登録ミス？
- 現状のアプリ対応: 同時間帯の科目は横に並べて表示

### 調査結果
- シラバスを比較すると**授業概要・到達目標・教科書・教員が全て同一**
- 科目番号が `02101036` と `02101036-A` で、同一モジュールを2科目コードに分けて登録している
- 021181(顔面・顎・口腔疾患)の方に「モジュール名：顎口腔医療」と記載あり
- → 実態は1つの授業。時間割上は重複として横並び表示するか、片方を非表示にするか要検討

### 全科目重複調査結果

**飯島さんの28科目内:** 重複は DEN.A309 / DEN.A308 の1組のみ。

**歯学部全102科目:** 89組の重複あり。ただし大半は異なる学年の科目同士。
- 100%一致: 学年混合セミナーⅠ/Ⅱ/Ⅲ (全8回一致、3科目とも同一)、顎口腔医療/顔面・顎・口腔疾患 (全27回一致)
- 部分重複: 異なる学年の科目が同じコマを使用しているケース（同一学生が両方履修することはない）
- → アプリはユーザーの履修科目のみ表示するので、飯島さんの A308/A309 以外は実害なし

### モジュール分割パターン（歯学部全体で15組）
医歯学系では1つのモジュールを複数の科目コード（-A, -B, -C...）に分割して登録するのが一般的。
飯島さんの28科目中、7組のモジュールに該当する科目が含まれている。
時間が完全重複するのは 02101036（A308/A309）の1組のみ。他は時間は別。

### 全科目調査（医学部+歯学部 計570科目、47モジュールグループ）
同一モジュール内で80%以上の時間重複があるもの: 5組
- 021180/021181 顎口腔医療（100%一致、27回）← 飯島さんに該当
- 021591/021592 選択コース1-A/1-B（100%一致、5回）← 選択式
- 021618/021619/021622 選択コース2-A/2-B/2-E（100%一致、5回）← 選択式

選択コースは「同時間帯から1つ選ぶ」仕様なのでアプリ上問題なし。
**実質的に対処が必要なのは 021180/021181 の1組のみ。**

### 飯島さんに確認したいこと
- 021180(顎口腔医療) と 021181(顔面・顎・口腔疾患) は実際どう受講している？（1つの授業？片方だけ出席？）
