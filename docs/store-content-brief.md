# App Store 掲載コンテンツ制作依頼書

## 依頼概要

東京科学大学の学生向けキャンパスSNSアプリ「ScienceTokyo App」を App Store に申請します。
以下の掲載コンテンツの制作を依頼します。

---

## アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | ScienceTokyo App |
| 対象ユーザー | 東京科学大学（旧 東京工業大学・東京医科歯科大学）の学生 |
| カテゴリ | 教育 |
| 価格 | 無料 |
| プラットフォーム | iOS（iPhone / iPad） |

## 主な機能

1. **時間割・課題管理** — 大学LMS（Moodle）と自動連携。時間割表示、課題の期限管理、成績確認
2. **キャンパスSNS** — 学内限定の投稿・コメント・リアクション・ダイレクトメッセージ
3. **友達の居場所** — キャンパス内で友達の現在地をリアルタイム共有（完全オプトイン）
4. **キャンパスナビ** — キャンパス内スポット検索・マップナビゲーション
5. **すれ違い通信** — 近くにいる人とプロフィールカードを自動交換
6. **イベント管理** — 学内イベントの作成・参加・出欠管理
7. **セキュリティ** — パスコードロック・Face ID / Touch ID 対応

## 競合・差別化ポイント

- 大学公式アプリではなく**学生が作った学生のためのアプリ**
- Moodle と直接連携して**時間割・課題を自動取得**できるのはこのアプリだけ
- **学内限定SNS**なので安心して使える
- 位置情報共有は**完全オプトイン**でプライバシーに配慮

## 依頼内容

### 1. アプリ説明文（概要）

- **文字数**: 最大4,000文字（日本語）
- **目的**: App Store の商品ページに表示される説明文
- **トーン**: 親しみやすく、大学生に刺さる表現。堅すぎない
- **含めるべき内容**: 主な機能の紹介、対象ユーザーの明示、利用条件（学生アカウント必要）
- **注意**: 他アプリ名の言及禁止（Apple ガイドライン）

### 2. プロモーション用テキスト

- **文字数**: 最大170文字
- **目的**: App Store で概要の上に表示される短いキャッチコピー
- **注意**: 審査なしでいつでも変更可能なので、季節やイベントに合わせた訴求も可

### 3. キーワード

- **文字数**: 最大100文字（カンマ区切り）
- **目的**: App Store 検索でヒットさせるための検索キーワード
- **注意**: アプリ名に含まれる単語は不要（自動でインデックスされる）

### 4. スクリーンショット（最重要）

- **枚数**: 5〜6枚（最低2枚、最大10枚）
- **サイズ**: iPhone 6.5インチ（1242×2688px または 1284×2778px）
- **iPad**: 12.9インチ（2048×2732px）も必要
- **内容の優先順**:
  1. ホーム画面（第一印象）
  2. 時間割（メイン機能）
  3. 課題一覧（実用性）
  4. SNSフィード（コミュニティ感）
  5. マップ / ナビ（ビジュアル映え）
  6. DM / チャット（SNS補強）
- **スタイル指定**:
  - デモデータで撮影（個人情報の映り込み禁止）
  - ライトモードまたはダークモードで統一
  - テキストキャプション（各画面の機能を一言で説明するオーバーレイ）をつけるかは任意
- **Apple ガイドライン**: 実際のアプリ画面であること。過度な装飾・誤解を招く表現は禁止

### 5. アプリアイコン

- **サイズ**: 1024×1024px PNG（角丸なし、Apple が自動で角丸にする）
- **現状**: 既存アイコンあり（資料/icon.jpg）。そのまま使うか改善するか判断
- **注意**: アルファチャンネル（透過）禁止
